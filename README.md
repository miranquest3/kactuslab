KactusLab

A React + Vite website scaffolded with Tailwind CSS and React Router.

Quick Start (Windows PowerShell):
1. npm install
2. npm run dev

Build and Preview:
1. npm run build
2. npm run preview

Replace Login Background Image:
- Place your image at: src/assets/bg.jpg (current)
- You can also use src/assets/bg.png — if you do, update the import in `src/pages/Login.jsx`.

Branding:
- Update title in index.html
- Update Navbar brand text in src/components/Navbar.jsx

Notes:
- Login is front-end only using localStorage for demo.
